# مجلد الأيقونات

ضع الأيقونات التالية في هذا المجلد:

## الأيقونات المطلوبة:

### Windows:
- `icon.ico` - أيقونة Windows (256x256 pixels)

### macOS:
- `icon.icns` - أيقونة macOS 

### Linux:
- `icon.png` - أيقونة Linux (512x512 pixels)

## إنشاء الأيقونات:

يمكنك استخدام أي صورة وتحويلها للصيغ المطلوبة باستخدام:

### أدوات أونلاين:
- https://convertio.co/png-ico/
- https://cloudconvert.com/png-to-icns
- https://favicon.io/favicon-converter/

### أدوات سطح المكتب:
- GIMP (مجاني)
- Photoshop
- Online ICO converter

## الأحجام المقترحة:
- **Windows ICO**: 16x16, 32x32, 48x48, 64x64, 128x128, 256x256
- **macOS ICNS**: 16x16, 32x32, 128x128, 256x256, 512x512, 1024x1024
- **Linux PNG**: 512x512 (أو أكبر)

## ملاحظة:
إذا لم تضع أيقونات، سيستخدم Electron الأيقونة الافتراضية.
